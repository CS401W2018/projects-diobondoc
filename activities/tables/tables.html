Engagement Questions

When do you use a table?

- When you have data that needs to be organized. Example excel spreadsheet and you want to put that in a webpage. Maybe players on a favorite team, stats, position, schedule, and etc.

When do you not use a table?

- Not use table to layout elements on page.

When do you need to do extra work for table accessibility?

- When dealing with more tables and a lot of headers. This will make the table a lot more easier to read.

Why don't we use the border attribute?

- It will get a validation error. This border will override css.

What is the easiest way to color every other row a different color?

- Use pseudo class to make it easier.
