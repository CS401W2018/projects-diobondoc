Internet

Internet is a network of computers and servers that communicate and connects to each other

Web

Web can be accessed via internet through the use of browsers to access different content

How are the internet and Web Different?

Internet is responsible to connect computers and servers to each other. While web is one of the services that can be accessed using the internet.

ASCII and Unicode

ASCII encodes digits, letters and some symbols. Unicode is capable to encode special texts and symbols from different languages

What is UTF-8

It stands for Unicode Transformation Format. UTF-8 is a method to encode characters and it usually uses 8-bit units

URL

URL is a unique address to locate resources on the internet

Domain name

A domain name is the physical name of the website

HTTP

HTTP stands for Hypertext Transfer Protocol. These are rules on how web browsers and servers connect to each other

HTTPS

HTTPS stands for Hypertext Transfer Protocol Secure. This is just like HTTP but it is a more secure version

IP Address

IP address is a unique identifier assigned to a device on a network

DNS

DNS stands for Domain Name System. This translates domain names readable by humans to a machine readable IP address

W3C Recommendations

W3C are the best practices of web development

XML, XHTML, HTML, HTML5

XML is a markup language that stores transmit and reconstruct data. XHTML is like XML but has extensions of HTML. HTML is the standard coding language used to build and structure contents on the web. HTML5 is the newest version of HTML. It supports videos and audio and other additional features.
